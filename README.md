# PuppyTrack

Dog bathroom training tracker for Android and iOS.

Log every bathroom trip, track your dog's feeding rhythm, and get predictions for the next break — built on real data instead of breed averages.

## Features

- One-tap logging — pee, poop, or both; success or accident
- Feeding rhythm — water→pee and food→poop intervals calculated from your dog's own records
- Smart predictions on the home screen
- Accident-free streak tracking
- Charts: hourly distribution, daily trends, time-of-day breakdown
- Full history with filters and delete
- Morning and night reminder notifications
- Multiple dog profiles
- Training levels (Beginner → Master)
- English and Turkish

## Tech Stack

- React Native + Expo SDK 56
- expo-router (file-based navigation)
- Supabase (auth + database)
- TypeScript
